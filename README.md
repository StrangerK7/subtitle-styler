# Subtitle Styler

DaVinci Resolve plugin for automated subtitle styling with Indian font support.

## Goals
- One-click apply professional subtitle styles
- Devanagari/Hindi font support
- Template library (corporate, reel, podcast, Hinglish badge style)
- Word-by-word animations
- Karaoke-style highlighting
